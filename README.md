## 📖 项目简介

这是一个用 Python 编写的 12306 抢票脚本，是我在学习编程过程中的第一个完整小项目。当时和 AI 一起折腾了好几天，从接口分析到自动化模拟请求，一步步实现了自动登录、余票查询、自动提交订单等核心功能。

**⚠️ 注意**：由于 12306 官方对自动化脚本的限制，该项目目前**已无法正常使用**，仅作为学习纪念保留。

---

## ✨ 功能特性

- 🔐 12306 账号自动登录（含验证码识别）
- 🔍 实时余票查询与监控
- 🚀 自动提交订单
- 📧 抢票成功邮件通知
- ⏰ 定时抢票功能

---

## 🛠️ 技术栈

- **Python 3.7+**
- Requests —— 网络请求
- BeautifulSoup4 —— HTML 解析
