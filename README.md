## 📖 项目简介

这是我学习 Python 自动化时写的一个实验项目，目标是尝试用 Selenium 模拟人工操作 12306 网站进行车票查询和预订。

**当前状态**：
- ✅ 可以自动打开浏览器并访问 12306
- ✅ 可以自动填充出发站、到达站、日期
- ✅ 可以自动点击查询按钮并循环刷新余票
- ✅ 可以识别有票的车次并尝试点击预订
- ⚠️ 登录需要手动操作
- ❌ 后续预订流程未完整实现

**为什么说"未完成"**：
由于 12306 网站反爬策略较强，以及个人时间原因，该项目仅完成了基础框架和查询模块，完整的自动下单功能尚未实现，且由于12306的反自动化脚本导致实际上已经无法实现自动抢票。现在仅作为一个学习纪念保留。
